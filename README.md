### BDX-R in mjlab 

## Quickstart

Get the robot description:

# Enter the repository
```bash
cd bdx_r_mjlab
```

# Get the asset
```bash
curl -L -o bdxr_r_cad.tar.gz https://github.com/KaydenKnapik/BDX-R-CAD/archive/refs/heads/main.tar.gz
tar -xzf bdxr_r_cad.tar.gz -C src/bdx_r_mjlab/robots/bdx_r/ 
rm bdxr_r_cad.tar.gz
```
